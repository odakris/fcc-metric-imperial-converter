Metric / Imperial Converter

This is a web app that convert metric units into imperial units and vice versa.

{ initNum: 3.1, initUnit: 'mi', returnNum: 4.98895, returnUnit: 'km', string: '3.1 miles converts to 4.98895 kilometers' }

Example usage

[your_url]/api/convert?input=4gal [your_url]/api/convert?input=1/2km [your_url]/api/convert?input=5.4/3lbs [your_url]/api/convert?input=kg

Example return

{ initNum: 3.1, initUnit: 'mi', returnNum: 4.98895, returnUnit: 'km', string: '3.1 miles converts to 4.98895 kilometers' }
